Detailed Module Breakdown
Module 1: Cybersecurity Fundamentals & Threat Landscape
• Theory: CIA triad, types of threats, OWASP Top 10, threat actors
• Practical: Simulate malware infection, analyze phishing emails
• Outcome: Understand threat modeling, perform basic risk assessments
• Tools: MITRE ATT&CK, OWASP Juice Shop
• Certs Aligned: CompTIA Security+
Module 2: Networking & Protocol Security
• Theory: TCP/IP, ARP poisoning, DNS spoofing, VLANs, NAT
• Practical: Packet analysis, basic IDS configuration
• Outcome: Network traffic inspection, secure communication setup
• Tools: Wireshark, Cisco ASA Emulator, pfSense
Module 3: Linux & Windows Security Essentials
• Theory: File permissions, system hardening, user management
• Practical: Audit logs, detect privilege escalation
• Outcome: Harden operating systems, understand logs
• Tools: Auditd, Event Viewer, PowerShell Scripts
Module 4: Cryptography & Quantum-safe Security
• Theory: AES, RSA, Elliptic Curve, Quantum threats
• Practical: Encrypt/decrypt files, SSL cert creation
• Outcome: Data encryption skills and PKI concepts
• Tools: OpenSSL, Keytool
• Certs: Security+ crypto section
Module 5: Ethical Hacking & Web Application Security
• Theory: Reconnaissance, SQLi, XSS, CSRF, Buffer Overflow
• Practical: Exploit web apps, capture flags
• Outcome: Identify and exploit web vulnerabilities
• Tools: Metasploit, Burp Suite, OWASP ZAP
• Certs: CEH v12
Module 6: IoT & Embedded Device Security
• Theory: IoT architecture, attack surfaces, device authentication
• Practical: Hack Raspberry Pi, BLE spoofing
• Outcome: Secure IoT systems and analyze embedded vulnerabilities
• Tools: Raspberry Pi, Arduino, Flipper Zero
Module 7: Cloud Security & DevSecOps
• Theory: IAM, Shared Responsibility, Container Security
• Practical: Configure cloud roles, scan code
• Outcome: Cloud configuration and DevSecOps basics
• Tools: AWS Console, Docker, GitHub, OWASP Dependency-Check
• Certs: AWS Certified Security – Foundational
Module 8: SOC Tools & Threat Intelligence
• Theory: SOC roles, log aggregation, IOC analysis
• Practical: Build SIEM dashboards, threat hunting
• Outcome: Detect intrusions, analyze incidents
• Tools: Splunk, ELK Stack, VirusTotal, Shodan
• Certs: Cisco CyberOps Associate
Module 9: AI-Driven Security & Zero Trust Architecture
• Theory: AI in anomaly detection, Zero Trust principles
• Practical: Create basic ML- based detection
• Outcome: Apply AI to security, design Zero Trust flow
• Tools: Jupyter, IBM Watson demos
Module 10: Red vs Blue Team Simulation
• Practical Only:
o Red: Phishing attack, reverse shells, lateral movement
o Blue: SOC alert response, log correlation
• Outcome: Offensive and defensive coordination
• Tools: Kali Linux, LAN Turtle, Splunk
Module 11: Digital Forensics & Incident Response
• Theory: Chain of custody, forensic triage, IR planning
• Practical: Memory analysis, deleted file recovery
• Outcome: Execute incident response plan
• Tools: FTK Imager, Autopsy, Volatility
Module 12: Capstone Project
• Goal: Simulate securing a company infrastructure or responding to a
breach
• Tasks: Network hardening, attack simulation, write IR report
• Outcome: Demonstrate holistic cybersecurity application
• Presentation to Evaluation Panel
